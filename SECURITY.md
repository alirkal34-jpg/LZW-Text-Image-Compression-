# Security Policy

## Supported Versions

The project currently supports security fixes on the latest default branch.

## Reporting a Vulnerability

Please report security vulnerabilities privately by opening a private security
advisory on GitHub (if enabled) or contacting the maintainers directly.

When reporting, include:
- A clear description of the issue
- Reproduction steps or proof of concept
- Impact assessment
- Suggested remediation (if available)

We will acknowledge reports as quickly as possible and work on a fix.
